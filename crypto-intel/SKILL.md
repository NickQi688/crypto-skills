# 币圈价值信息情报官

从加密货币资讯中筛选出高价值投资信息。

---

## 描述

自动调用 Grok API 抓取过去 N 小时的币圈高价值信息，包括价格异动、爆仓大户、项目动态、监管政策、链上数据。

---

## 使用方法

```
帮我抓取过去4小时的币圈资讯
```

或者带参数：

```
抓取币圈资讯:
- 时间范围: 12小时
- 最大条数: 20条
```

---

## 配置

在使用前，需要设置 Grok API 配置。

编辑配置文件：`.env`

```bash
GROK_API_KEY=your_api_key_here
GROK_API_ENDPOINT=https://ai.a9.bot/v1
GROK_API_MODEL=grok-4.20-beta
```

---

## 筛选标准

✅ **价格异动**: 单日涨跌 > 5%，或突破关键点位（$60K、$65K 等）
✅ **爆仓大户**: 单笔爆仓 > $1M，或 24H 总爆仓 > $10M
✅ **项目动态**: 融资 > $10M，或一线交易所（币安、Coinbase）上币
✅ **监管政策**: 影响市场的重要政策变动（SEC、各国政府）
✅ **链上数据**: ETF 资金流向 > $50M，或鲸鱼大额转账

---

## 输出格式

按类型分组，每条包含：
- 标题（吸引眼球）
- 时间
- 内容摘要（3-5句话）
- 影响/分析（简短）

---

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| timeframe | 4h | 抓取时间范围 |
| maxResults | 15 | 最大输出条数 |
| saveFile | true | 是否保存到文件 |

---

## 工作流程

1. 读取 .env 配置
2. 调用 Grok API（支持 SSE 流式响应）
3. 解析并格式化输出
4. 保存到文件（可选）

---

## 依赖

- Node.js (运行 fetch 脚本)
- Grok API Key
- https 模块（内置）
